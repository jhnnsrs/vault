Scheduler​

Scheduler try to create a provision mapping from a reservation.

Image you specify a reservation like this

Provide Maximum Intensity Node and paralleize it using preferably five instances, preferably again on on agent

ImageJ implements this Task and allows for one task per connected agent. Napari implements this Task and allows for one task per connected agent. Cluster implements this Task and allows for two task per connected agent.
